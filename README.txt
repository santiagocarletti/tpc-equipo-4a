# Restaurant Ordering and Order Management System

Web application developed in C# with ASP.NET WebForms for managing restaurant orders in a gastronomic environment, focused on the cashier role.

The system allows creating complete orders by combining individual products and configurable combos, editing each item independently, and storing the final order in a relational database.

---

## Description

The system allows a cashier to:

* Build orders with individual products and combos
* Configure each item (ingredients, extras, and comments)
* Register the final order exactly as requested by the customer

The application addresses the need to:

* Take orders in a structured way
* Handle combos with configurable products by group
* Avoid conflicts between identical items within the same order
* Correctly persist each order and its items

Academic project focused on business logic, data persistence, and state management within a realistic workflow.

---

## Technologies Used

* C#
* ASP.NET WebForms
* SQL Server
* ADO.NET
* HTML / CSS
* Bootstrap
* JavaScript (basic)

---

## Main Features

* Product and combo management
* Grouping of products by category (main course, drink, etc.)
* Order taking from a cashier interface
* In-memory shopping cart (Session-based)
* Individual editing of each order item
* Handling of optional ingredients and extras
* Comments per product or combo
* Order and item persistence in the database
* Unique item identification to avoid mixing identical products

# Sistema de Comandas y Toma de Pedidos

Aplicación web desarrollada en C# con ASP.NET WebForms para la gestión de pedidos (comandas) en un entorno gastronómico, orientada al rol de cajero.

Permite armar pedidos completos combinando productos individuales y combos configurables, editar cada ítem de forma independiente y registrar la comanda final en una base de datos relacional.

---

## Descripción

El sistema permite a un cajero:
- Armar pedidos con productos sueltos y combos
- Configurar cada ítem (ingredientes, extras y comentarios)
- Registrar la comanda final respetando exactamente lo solicitado por el cliente

La aplicación resuelve la necesidad de:
- Tomar pedidos de forma estructurada  
- Manejar combos con productos configurables por grupo  
- Evitar conflictos entre ítems iguales dentro de un mismo pedido  
- Persistir correctamente cada comanda y sus ítems  

Proyecto académico enfocado en lógica de negocio, persistencia de datos y manejo de estado en un flujo realista.

---

## Tecnologías utilizadas

- C#
- ASP.NET WebForms
- SQL Server
- ADO.NET
- HTML / CSS
- Bootstrap
- JavaScript (básico)

---

## Funcionalidades principales

- Gestión de productos y combos  
- Agrupación de productos por grupo (principal, bebida, etc.)  
- Toma de pedidos desde pantalla de cajero  
- Carrito de pedidos en memoria (Session)  
- Edición individual de cada ítem del pedido  
- Manejo de ingredientes opcionales y extras  
- Comentarios por producto o combo  
- Registro de comandas e ítems en base de datos  
- Identificación única de ítems para evitar mezclas entre productos iguales
